# LineUs_SVG
Simple SVG plotter for the LineUs drawbot made with Processing 4.4.10 and Geomerative Plugin.

![Image of the application](https://github.com/ixd-hof/LineUs_SVG/raw/master/LineUs_SVG.png)

## Usage:
You can simply open the code in [Processing 4.4.10](https://processing.org/download/), install the [Geomerative] plugin (via Contribution Manager) and run the sketch. Since this plotter is dependent on the SVG functions of Geomerative (flattening and polygonizer) which was finaly updated to support version 4.4.10


_Application interface / keys / shortcuts:_
* drawbot address: a (usually line-us.local in wifi and 192.168.4.1 in setup mode)
* open SVG: o
* zoom: +/-
* move: arrow keys <>
* rotate: r
* show lines: l
* connect Line-Us: c
* plot drawing: p
* hide menu: h
